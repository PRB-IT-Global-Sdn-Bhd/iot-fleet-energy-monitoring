# IoT Solutions — Fleet & Transport / Energy Monitoring

> **PRB IT Global Sdn Bhd** | Custom Software for SME Engineering & Construction Clients (Malaysia)

## Overview

A comprehensive **IoT-powered Fleet and Energy Monitoring System** for engineering and construction companies. The platform provides real-time tracking of vehicles and heavy machinery, alongside energy consumption monitoring for sites and facilities — driving cost efficiency and operational excellence.

## Key Features

| Feature | Description |
|---|---|
| GPS Fleet Tracking | Real-time vehicle and machinery location on map |
| Driver Behaviour | Speed, harsh braking, and idle time monitoring |
| Fuel Monitoring | Fuel level sensors with consumption analytics |
| Maintenance Alerts | Predictive maintenance based on engine hours/mileage |
| Energy Metering | Smart meters for electricity consumption per zone |
| Carbon Reporting | GHG emissions tracking for ESG compliance |
| Route Optimisation | AI-driven route planning for delivery and logistics |

## Technology Stack

- **Hardware:** GPS trackers, OBD-II dongles, smart energy meters, IoT gateways
- **Protocol:** MQTT, 4G/LTE, CAN Bus
- **Backend:** Python (FastAPI), Node.js
- **Database:** InfluxDB + PostgreSQL
- **Maps:** Google Maps API / Mapbox
- **Cloud:** AWS IoT Core / Azure IoT Hub

## Target Users

Fleet managers, logistics coordinators, facility managers, and sustainability officers.

## Getting Started

```bash
git clone https://github.com/PRB-IT-Global-Sdn-Bhd/iot-fleet-energy-monitoring.git
cd iot-fleet-energy-monitoring
pip install -r requirements.txt
python main.py
```
